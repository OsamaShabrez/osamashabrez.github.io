Scanner APIs

  1. quaggaJS - Barcode Reader 92kb (doesn't support QR Code)
    - Takes array of readers as input but more readers = more false positives
    https://github.com/serratus/quaggaJS
  2. jsQR - QR Code Reader - 248KB not stable, false positives and decoding issues
    https://github.com/cozmo/jsQR
  3. zxing-js 205KB (barcode and QRcode scanning)
    - Served via unpkg: https://unpkg.com/@zxing/library@latest, current version: 0.15.2
    - Can decode from video stream and images without configuring reader details. Supports Code128, EAN-13, ITF and QR codes.
    - ZXing can also generate QR Code SVG

Camera APIs

  1. Navigator.MediaDevices
    MediaDevices Compatibility - No IE Support
    Native APi, works well on all modern browsers, can be extended to use as video recorder.

  1. SignaturePad
    Served via: https://szimek.github.io/signature_pad/js/signature_pad.umd.js
